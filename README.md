# SWV-based SOC proxy sensing platform

This repository contains the processed datasets and R scripts associated with the manuscript:

**Development of a portable electrochemical sensing platform for soil organic carbon proxy measurements based on square-wave voltammetry**

## Repository contents

- `data/processed/`: processed electrochemical records, soil covariates, extracted SWV descriptors, and model outputs.
- `data/figure_source_data/`: source data used to generate manuscript figures.
- `R/`: R scripts used for data processing, figure generation, feature extraction, learning-curve simulation, and exploratory Random Forest modeling.
- `figures/`: selected figures generated from the processed datasets.
- `docs/`: variable dictionaries and supporting documentation.

## Scope

The repository supports reproducibility of the analyses, tables, and figures reported in the manuscript. The data correspond to processed electrochemical measurements and associated soil information used for exploratory SOC proxy inference.

## Software availability

The custom executable software developed for the acquisition and processing workflow is not included in this repository because it remains part of an ongoing hardware–software development process and is still under technical validation.
